<h1 style="text-align: center;"> Instagram Mapping </h1>

<h2> Project Summary </h2>
  
  <p> The aim of this project is to provide various automations on Instagram using Python programming language and Selenium library. Selenium kütüphanesi Selenium is a         tool that allows you to open a browser such as chrome or firefox on the screen with the help of a driver you install on your computer and run all the operations          you want like a real person with the help of a programming language. </p>
 
  
  <p> 
  What the program can do : 
  <ul>
    <li> extract following list (users & others) </li>
    <li> extract follower list (users & others) </li>
    <li> compare following/followers </li>
    <li> unfollow user(s) </li>
    <li> message user(s) </li>
    <li> get profile informations </li>
    <li> download account posts and zip </li>
  </ul>

  <b> It is still in development so if you have any feature you want to see in the program please write in discussion </b>
  </p>

<h2> Set-Up </h2>

- Clone the repo 
```
git clone https://github.com/Arslanex/Instagram-Mapping
```
- Install requirements
```
pip install -r requirements.txt
```
- Download webdriver

<h2> How To Use </h2>
<ol>
<li>Create a main.py file</li>
<li>Enter your informations to userinfo.py</li>
<li>import Instagram_Bot file </li>
<li>With Instagram_Bot file create a Bot object </li>
<li>Call the functions that will do what you want</li>
<li>Run main.py file</li>
</ol>
