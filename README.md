<h1 style="text-align: center;"> Instagram Mapping </h1>

<h3> Project Summary </h3>
  
  <p> The aim of this project is collect information from your personal instagram account and create a report about it. 
  This report eill contain valuable information about users. Such as personal information from bio (collage, high school, etc.),
  downloading  accounts post and stories then zip them, creating links between users that determines proximity of other users, etc.
  With this project you can also check your account as well. For example you can compare your followers and following lists. In result 
  program will return 3 lists ; 
  
  <ul>
    <li>Followers - who follows you but you don't follow back</li>
    <li>Mutual - who follows you and you follow back</li>
    <li>Following - who you follow but don't follow you back
  </ul>

  This project also automate some of the tasks in Instagram. The tasks that the program can do it is down below :

  <ul>
    <li> message user </li>
    <li> message users </li>
    <li> unfollow user </li>
    <li> unfollow users </li>
  </ul>

  <b> It is still in development so if you have any feature you want to see in the program please write in discussion </b>
  </p>
